# TCC 2026 – Repositório Exemplo

Este repositório serve como **modelo oficial** para os alunos da turma 3F 2026 – LEOS organizarem todos os materiais relacionados ao Trabalho de Conclusão de Curso (TCC).  
Aqui os alunos podem visualizar como estruturar corretamente um projeto completo no GitHub.

---

## 🎯 Objetivo deste repositório

Demonstrar como um grupo deve organizar:

- Pesquisas de mercado  
- Escopo do TCC  
- Estudos de viabilidade  
- Relatórios de acompanhamento  
- Comentários e issues de professores (orientador e apoiador)  
- Protótipos e mockups  
- Especificações técnicas  
- Código‑fonte  
- APIs  
- Documentação  
- Apresentações  
- Versão final do TCC
